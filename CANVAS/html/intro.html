Introdução

O que é o Canvas?

Canvas é um elemento HTML usado para desenhar gráficos e é juntamente usado com JavaScript.

Ele é usado apenas para uso com gráficos e imagens e possui vários métodos, via Javascript, para desenhar formas geométricas, textos com ou sem animação e muito mais.

O que o Canvas pode fazer?

Desenhar textos

O elemento canvas pode ser utilizado para fazer textos coloridos, com ou sem animação.

Desenhar gráficos

O elemento canvas tem vários recursos para apresentação de gráficos e galerias de imagens.

Pode ser animado

Objetos no canvas poder se mover e é capaz de fazer de simples a complexas animações.

Pode ser interativo

Canvas pode responder a eventos em JavaScript e também a eventos acionados pelo usuário como um clique do mouse, apertar uma tecla no teclado, movimentos captados por sensores e muito mais.

Pode ser usado para criação de jogos

Os métodos de animações do canvas oferecem grandes possibilidades para as aplicações em jogos HTML.

Canvas e id's

Todo elemento canvas deve ter um id para poder ser relacionado pelo JavaScript. Também é necessário informar a largura, width, e altura, height, do elemento canvas.

Por padrão o elemento canvas não tem bordas, border.
