Passo a passo para instalar e configurar o servidor LAMP

Configurando o servidor Apache2

Instalação

Para instalar o LAMP digite o comando abaixo no seu terminal:

sudo apt-get install lamp-server^

O símbolo (^) faz parte do comando.

Primeiro teste no Apache

Logo após a instalação execute o código abaixo:

sudo /etc/init.d/apache2 restart

Agora acesse o endereço http://localhost no seu navegador.

Se tudo der certo vocẽ verá a página escritoIt works!

Como exibir sua página no localhost

Para começar copie a configuração do site padrão:

sudo cp /etc/apache2/sites-availabe/000-default.conf /etc/apache2/sites-available/meusite.conf

Agora edite o arquivo com seu editor de texto pelo terminal com o seguinte comando:

sudo pluma /etc/apache2/sites-available/meusite.conf

Dentro do arquivo, na área DocumentRoot indique o caminho de onde o documento raíz do seu site.
por exemplo:

DocumentRoot /home/user/meusite_html

Agora é só salvar o arquivo e sair.

Ativando seu site

Agora é necessário que a página padrão seja desativada e a sua página seja ativada no lugar dela.
Para isso basta digitar o comando abaixo:

sudo a2dissite 000-default && sudo a2ensite meusite

Depois é necessário reiniciar o servidor:

sudo /etc/init.d/apache2 restart

Agora é só acessar o localhost e sua página estará funcionando.
