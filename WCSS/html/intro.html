Porque usar CSS?

O que é CSS?

CSS é uma abreviação para Cascading Style Sheets, em português, Folhas de Estilos em Cascatas. O CSS estiliza sua página html, passando informação visual.

O CSS facilita essa estilização porque um único arquivo .css pode alterar todas as páginas de uma única vez sem precisar abrir cada arquivo .html e mudar um a um.

Como inserir CSS no HTML

Inline CSS

O Inline CSS é usado dentro do arquivo html, diretamente no elemento onde se deseja usar um estilo. Vamos ver um exemplo de como usar um Inline CSS:

      
      <h1 style="background-color:#00f;">

Nesse exemplo eu usei o CSS diretamente dentro da tag que está dentro de um documento html. Essa forma de CSS deve ser usada apenas em casos extremos. Evite ao máximo.

Internal CSS

O Internal CSS ou, CSS interno, é usado dentro das tags <head> e é comum de ser usado entre os programadores WEB. Abaixo vou mostar um exemplo:


      <head>
        <style>
        h1 {
          background-color: #00f;
        }
        </style>
      </head>

Esse tipo de CSS é usado com frequência mas ainda assim só estiliza a página onde ele estiver.

External CSS

O External CSS é um arquivo CSS que, como o próprio nome já diz, está do lado de fora do documento em .html. Esse é o tipo mais comum de ser usado e também o mais apropriado, pois facilita a manutenção dos estilos em vários arquivos .html, poupando trabalho de abrir-los um a um para modificar cada página. Abaixo um exemplo de como se escreve esse tipo de CSS:


      h1 {
        background-color: #00f;
      }

Dentro do arquivo .css não precisa colocar as tags <style>, basta colocar qual tag vai ser modificada e seus atributos, valores e propriedades.

Para mostrar para o documento em .html onde está o arquivo .css que vai estilizá-lo, usa-se a tag <link> para isso dentro das tags <head>.


      <head>
        <link rel="stylesheet" type="text/css" href="main.css"/>
      </head>
