Usando suas próprias fontes

Para usar suas próprias fontes dentro de uma página html é bem simples. Primeiro faça uma pasta dentro do local onde estão seus arquivos. No meu caso eu criei um diretório chamado font dentro do local onde está o meu site.

Com a pasta feita, coloque os arquivos de fontes que deseja usar. Caso não tenha fontes disponíveis no seu computador, encontre fontas no site Font Library ou no Dafont, que são fontes gratuitas para uso pessoal.

Agora dentro do arquivo .css da página que você deseja personalizar você escreve o seguinte código:


      @font-face {
        font-family: SuaFonte;
        src: url(../font/SuaFonte.ttf);
      }

Na área font-family você vai desginar um nome para a fonte, não precisa ser o mesmo do nome original, pois esse nome é só para ser usado como referência dentro do arquivo .css

A área src indica onde está a fonte que você quer usar, no exemplo url(../font/SuaFonte.ttf) eu estou indicando que minha fonte está um diretório acima do diretório onde está localizado esse arquivo .css mas caso sua fonte ou .css esteja em outro local, o caminho pode ser diferente de como está aqui.
